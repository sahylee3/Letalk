# LeTalk — 대화에 맛을 더하다

최신 뉴스 기반 스몰토크 & 토론 주제 생성기.
순한맛부터 매운맛까지, 대화의 깊이를 선택하세요.

## 배포 방법

### Vercel 배포 (추천)
1. 이 저장소를 GitHub에 업로드
2. [vercel.com](https://vercel.com) 가입
3. "Import Project" → GitHub 저장소 선택 → Deploy

### 주간 업데이트
`app/page.js` 파일의 `TRENDING` 배열을 수정하면 주제가 업데이트됩니다.

## 기술 스택
- Next.js 14
- React 18
- Tailwind CSS
- Pretendard 폰트
